# My Portfolio Site

A clean, modern portfolio site built with HTML, CSS, and vanilla JavaScript.

## Features

- 📱 Fully responsive design
- ✨ Modern and minimalist aesthetic
- 🎯 Smooth navigation and interactions
- 🎨 Customizable color scheme
- ⚡ Fast and lightweight

## Sections

- **Hero** — Eye-catching introduction
- **About** — Skills and background
- **Projects** — Showcase your work
- **Contact** — Easy ways to reach you

## Getting Started

### Customization

1. **Update your info**
   - Replace "Gethin" with your name
   - Update email and social links in the contact section
   - Add your actual project descriptions

2. **Personalize colors**
   - Edit the CSS variables in `styles.css`:
     ```css
     --primary-color: #2563eb;
     --secondary-color: #1e40af;
     ```

3. **Add your projects**
   - Replace the placeholder project cards with your real projects
   - Add links to GitHub repos or live demos

## Deployment

This site is automatically deployed to GitHub Pages. Just push your changes to the `main` branch!

View it live at: `https://gethin101.github.io`

## Tips

- Keep your portfolio up-to-date with recent projects
- Use high-quality descriptions of what you built
- Include links to GitHub repos and live demos
- Consider adding a blog section as you grow
- Monitor analytics to see where visitors come from
